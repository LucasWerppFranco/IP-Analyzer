## Project Purpose

This project was created to help students and professionals perform tedious IP calculations more quickly. In the future, I intend to evolve this project to include IPv6 or other functionalities.

## Running the Code

Compile the code:
```
cargo build --release
```

Run the code:
```
./target/release/meu_projeto
```
